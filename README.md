backend
Создает начинку базы ЦУНАМИ.
А.Как запустить:
1) Скачать пакет необходимых программ (WebStorm, MongoDB Compass, Postman)
2) Клонируем репозиторий с GitHub в WebStorm (выбираем в окне старта вкладку GitHub и вставляем ссылку проекта)
3) Качаем Node.JS
4) Указываем путь к папке с Node.JS через Edit configurations...(если не указана автоматически)
5) Устанавливаем необходимые библиотеки через терминал: npm install
6) Проверить наличие файла в корне: ".evn"
7) Запускаем сервер через терминал: npm start
B.Как настроить:
Настройка происходит в файле: ".evn"
Файл содержит 3 переменных:
1) DATABASE_URL - путь к mongoDB. Он содержит данные, занесенные в базу. Необходимо указать ссылку.
2) PORT - порт, который будет занимать сервер.
3) JWT_SECRET - любая строка. Используется для шифровки JWT токена.
